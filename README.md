## Introduction

- The dashboard anlayzes the data of almost 1000 patients admited to the hospital 
from 2021 to 2024. You can interactively filter by either condition(s) or 
treatment(s) to update the dashboard and view specific information about the
selected fields. Various insights can be concluded from this dashboard whether
about diseases, treatments, patients, hospital revenue and hospitalization 
periods.

- The Power BI file (.pbix) is recommended to experience the dashboard's
interactivity, and a pdf version is added for a quick view.
Data source : [Source](https://www.kaggle.com/datasets/devildyno/hospital-patient-records-jan-2021-july-2024)

## Instructions

- Control+click the "Condition" or "Treatment" buttons on the menu on
the left of the dashboard to filter by the selected button.
- You can use the slicer titled "Select Medical Condition" or "Select Treatment" to filter, control+click
if you want to select multiple fields.
- Scroll up/down or left/right the table to view more columns and rows.
